# gs-rest-service
Using spring & gradle.

# Spring
- It is a lightweight & open-source framework.
- It is said to be non-invasive.

Feature:
- POJO model
- AOP
- Loose coupling

Modules:
- Core
- Context
- DAO
- ORM
- AOP
- Web-MVC

MVC Architecture:
* M - Model: Database + ORM (Hibernate + MySQL)
* V - View (html, jsp, js)
* C - Controller (Servlets)
